Thank you for installing MatchPDF!

Install your license using a command prompt by simply running the executable:
C:\match_pdf.exe  
  
If a license was previously installed, a new one can be applied:
C:\match_pdf.exe --license

(Get a paid or trial license at MatchPDF.com)

See the UserGuide.pdf in your install directory for more info.
